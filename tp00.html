7 ¿a que llamamos elemento html? ¿cual es su anatomia?. explicar sus partes

llamamos elemeno html a la raiz de un documento HTML, El resto de elementos descienden de este elemento. la anatomia de un elemento de html se puede clasificar en 4 partes: una es el CONTENIDO, que se encuentra dentro de la etiqueta que hemos creado, despues el ELEMENTO que es todo el conjunto de la etiqueta de apertura y de cierre mas el contenido. etiqueta de apertura, consiste en el nombre del elemento encerrado por paréntesis angulares y la etiqueta de cierre es lo mismo que la etiqueta de apertura con la diferencia que lleva una barra antes del nomre y determina donde acaba el elemento

9 que hace la etiqueta p, i, strong, em, sub, u, br

el elemento p sirve para distribuir el texto en parrafos
la etiqueta i muestra el texto marcado con un estilo en cursiva o italica.
la etiqueta strong sirve para resaltar las palabras
la etiqueta em sirve para marcar con énfasis las partes importantes de un texto.
la etiqueta sub sirve para mostrar las letras mas pequeñas
la etiqueta u sirve para marcar con una linea una palabra
la etiqueta br sirve para separar los textos de los parrafos
